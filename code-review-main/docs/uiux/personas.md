//user Persona

 Developer / Student

Role:
A user who wants to review and improve their code using an AI-powered code review tool.

Goals:
- Identify mistakes in the code quickly.
- Improve code quality.
- Save time during debugging.

Pain Points:
- Debugging code takes too much time
- Difficult to identify errors quickly
- Lack of knowledge about best coding practices

Needs:
- Instant AI code analysis
- Clear explanation of mistakes
- Corrected version of the code


// AI Persona

 AI Code Reviewer

Role:
An intelligent assistant that analyzes the user's code and provides suggestions.

Responsibilities:
- Analyze the submitted code
- Detect syntax errors and logical issues
- Provide improvement suggestions
- Generate corrected code

Capabilities:
- Code analysis
- Error detection
- Code improvement suggestions
- Clean and corrected code output
